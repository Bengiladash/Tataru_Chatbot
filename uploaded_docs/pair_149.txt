https://www.csusb.edu/its/support/it-knowledge-base/detail?id=9459079570965e806575c944439f2d928e737c4541 Outlook - Forward as attachment (PC) From time to time a tech needs to see the full headers of a message to analyse the issue. The easiest method to do this is to forward the message as an attachment to the tech. To do this in Outlook, follow the steps below. Select the message. Make sure the Home tab is selected, then go to More -> Forward as Attachment. Compose the message the way you normally do and click the Send button when done.
