https://www.csusb.edu/its/support/it-knowledge-base/detail?id=948ffc8ca31fff8616d6354731950fb7042a1b827a How to set up DUO with a new phone number? Log into MFA Portal with you 9-digit ID number and password Select Need help? Select "I got a new phone" Authenticate with old device Click on “Add a device” Select DUO Mobile Enter your new phone number Verify your phone number Install the DUO mobile app and select Next Scan the QR code or choose to have an activation link emailed to you You are now connected via DUO
